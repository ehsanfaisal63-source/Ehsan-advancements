# **App Name**: Ehsan advanced website

## Core Features:

- Hero Section: An engaging introduction with visually appealing elements and a clear call to action.
- Firebase Authentication: Secure login and signup functionality using Firebase Authentication.
- Firestore Integration: Store and retrieve user details and other app data using Firestore.
- Protected Dashboard: A secure dashboard that displays user-specific data loaded from Firestore.
- Responsive UI: A user interface that adapts seamlessly to different screen sizes and devices.
- Smooth Animations: Subtle animations to enhance user experience.
- Code Deployment: The Next.js code ready to be deployed on Firebase Hosting

## Style Guidelines:

- Primary color: Deep Blue (#1A237E), reminiscent of Firebase's professional and modern aesthetic.
- Background color: Very dark desaturated blue (#0A0F28) to enhance the contrast of lighter UI elements, optimized for a dark color scheme.
- Accent color: Purple (#9C27B0) to complement the primary blue, adding a vibrant touch.
- Body and headline font: 'Inter', sans-serif, to ensure a clean and modern look, offering excellent readability.
- Use Google Material Icons for a consistent and recognizable visual language.
- Implement a Glassmorphism UI with gradient backgrounds to give a frosted glass effect, adding depth and sophistication.
- Incorporate subtle animations on page transitions and button hovers for a smooth user experience. and add ai features
